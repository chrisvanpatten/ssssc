Sexy Style Sheets with Sass

(and Compass)

What's the problem with CSS?

  • Static

    No programmatic capabilities

  • Repetition

    Constantly repeating key colors, values, bits of style

  • Vendor prefixes

    New features are prefixed, and often have varying syntaxes

  • Just to name a few...

How can Sass help?

  • Sass (and other preprocessors) extend and improve CSS

    They're technically "meta-languages"

  • Sass provides programmatic capabilities

    Variables, math, functions, basic scripting

  • Sass introduces new syntax

    Stop repeating yourself with nesting and extending

  • Sass compiles to CSS

    It replaces CSS in your workflow, but is compiled to CSS for production

Sass Features

  • Variables

  • Math

  • Helpers

  • Nesting

  • Mixins

  • Inheritance/Extending

  • Let's look at these features individually....

Variables

Input...

$blue: #3bbfce;
$margin: 16px;
$sans-serif: 'HelveticaNeue-Light', 'Helvetica Neue Light', 'Helvetica Neue', Helvetica, Arial, 'Liberation Sans', FreeSans, sans-serif;

div {
	color: $blue;
	font: 16px 'Museo Sans', $sans-serif;
	margin: $margin;
}

Variables

Output...

div {
  color: #3bbfce;
  font: 16px "Museo Sans", "HelveticaNeue-Light", "Helvetica Neue Light", "Helvetica Neue", Helvetica, Arial, "Liberation Sans", FreeSans, sans-serif;
  margin: 16px;
}

Math

Math

div {
	width: 300px - (8px + 8px);
	padding: 8px;
	margin: $margin;
}

Math with variables

$margin: 16px;

div {
	width: 300px - $margin;
	padding: $margin / 2;
	margin: $margin;
}

Math

Output...

div {
  width: 284px;
  padding: 8px;
  margin: 16px;
}

Nesting

Input...

table.hl {
	margin: 2em 0;

	td.ln {
		text-align: right;
	}
}

li {
	font: {
		family: serif;
		weight: bold;
		size: 1.2em;
	}
}

Nesting

Output...

table.hl {
  margin: 2em 0;
}
table.hl td.ln {
  text-align: right;
}

li {
  font-family: serif;
  font-weight: bold;
  font-size: 1.2em;
}

Nesting

Input...

$link-color: #0000FF;

a {
	color: $link-color;

	&:hover, &:visited:hover {
		color: darken($link-color, 10%);
	}

	&:visited {
		color: darken($link-color, 5%);
	}

	&:active {
		color: lighten($link-color, 10%);
	}
}
